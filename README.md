
# Reseller Platform Starter

Modern React + Supabase reseller platform starter template.

## Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router

## Features
- Login only authentication
- Role based dashboard
- Admin / Reseller / Customer layouts
- Beautiful modern dark UI
- Wallet cards
- Analytics cards
- Responsive sidebar

## Run

```bash
npm install
npm run dev
```
